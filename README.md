# 2077 Cyber Bank Terminal 

A Cyberpunk 2077 themed banking system prototype featuring desktop UI components built using Python's `tkinter` library.

## 🚀 Features
- **Dual Terminal Access:** Separate, secure login interfaces for Admin Control and regular Users.
- **Cyberpunk Aesthetics:** Styled with neon cyber-grid colors (Cyber-yellow, Cyan, and Magenta accents).
- **Core Banking Logic:** Full support for adding, searching, and deleting user records, alongside real-time balance modifications.
